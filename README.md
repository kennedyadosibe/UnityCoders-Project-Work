Project Description:
This is a simple web application created as part of the BCP104 Programming with Python course group project. The application was built using only HTML and CSS.
The purpose of the project is to demonstrate the ability to design and style a functional and user-friendly website. The web application includes multiple pages and features such as:
1. A homepage with navigation links.
2. A content section (for example: teams, services, or members).
3. A styled layout with CSS for colors, fonts, and positioning.
4. A responsive design that adjusts to different screen sizes.


**Technologies Used:**
HTML for structure
CSS for styling
Git and GitHub for version control
Free hosting service


**Deployment:**

https://kennedyadosibe.github.io/UnityCoders-Project-Work/

How to Run the Project Locally:

1. Clone or download the project files from GitHub.
2. Open the project folder.
3. Double-click the main HTML file (index.html) to open it in a web browser.
